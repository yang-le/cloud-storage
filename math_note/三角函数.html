三角函数：定义和图像

以下考虑圆心在坐标原点的单位圆(半径为1)：

圆周率
圆的周长与其直径的比值，记号为$\pi$。
弧度
单位圆上，长度为1的弧所对的圆心角的大小为1弧度。
对于单位圆来说，其周长为$2\pi$，因此周角为$2\pi$弧度，平角为$\pi$弧度，直角为$\pi \over 2$弧度。

圆弧是有方向的，据此可对弧度规定正负。
规定逆时针(CCW：counter clock wise)旋转形成的弧所对应的弧度为正；顺时针(CW:clock wise)旋转形成的弧所对应的弧度为负。

设有一动点T在单位圆上运动，定义自变量$\theta$为坐标点$S=(1,0)$，坐标原点O,动点T所组成的角$\angle SOT$的弧度值。
点S称为起始点。动点T会随着时间的变化而变化，在某时刻t，点T的位置记为$(x,y)$。

正弦
点T的纵坐标值,记做$\sin\theta$。即$\sin\theta = y$
余弦
点T的横坐标值,记做$\cos\theta$。即$\cos\theta = x$
正切
定义为${y \over x} = {\sin\theta \over \cos\theta}$,记做$\tan\theta$。

另外还有三个三角函数分别是以上三者的倒数：

余割
$\csc\theta = {1 \over \sin\theta} = {1 \over y}$
正割
$\sec\theta = {1 \over \cos\theta} = {1 \over x}$
余切
$\cot\theta = {1 \over \tan\theta} = {\cos\theta \over \sin\theta} = {x \over y}$

三角函数：求值

$\theta$ 0 $\pi \over 4$ $\pi \over 2$ $3\pi \over 4$ $\pi$ $5\pi \over 4$ $3\pi \over 2$ $7\pi \over 4$ $2\pi$
$\sin\theta$ 0 $\sqrt2 \over 2$ 1 $\sqrt2 \over 2$ 0 $-{\sqrt2 \over 2}$ -1 $-{\sqrt2 \over 2}$ 0
$\cos\theta$ 1 $\sqrt2 \over 2$ 0 $-{\sqrt2 \over 2}$ -1 $-{\sqrt2 \over 2}$ 0 $\sqrt2 \over 2$ 1

观察上表，函数值关于$\pi$的对称性可以用周期性和奇偶性解释。即 $$\sin(\theta) = \sin(\theta \pm 2\pi)$$ $$\cos(\theta) = \cos(\theta \pm 2\pi)$$ $$\sin(-\theta) = -\sin(\theta)$$ $$\cos(-\theta) = \cos(\theta)$$ 例如 $\sin({7\pi \over 4}) = \sin(-{\pi \over 4}) = -\sin({\pi \over 4}) = -{\sqrt2 \over 2}$
对于两个相加等于$\pi \over 2$的角来说，可以从两个角度来看。
角度1：记$\alpha = \angle SOT$,$\beta = \angle TOE$，其中S点的坐标为(1, 0)，E点的坐标为(0, 1)，T点位于两者之间，坐标为(x,y)。显然有$\alpha + \beta = {\pi \over 2}$
角度2：对调x轴和y轴，这时S点的坐标变为(0, 1)，而E点的坐标变为(1, 0)，T点的坐标则相应变成了(y,x)。仍然有$\alpha + \beta = {\pi \over 2}$
从角度1来看，$\sin(\alpha) = y$。但从角度2来看，$\cos(\beta) = y$（注意这时y是横坐标值）。所以我们有 $$\sin(\alpha) = \cos(\beta) = \cos({\pi \over 2} - \alpha)$$ 类似地，我们还有 $$\cos(\alpha) = \sin(\beta) = \sin({\pi \over 2} - \alpha)$$ 下面我们来解释另一种对称性，即关于$\pi \over 2$的对称性
利用上面得出的式子,我们有$\cos(\pi + \theta) = \cos({\pi \over 2} - (- {\pi \over 2} -\theta)) = \sin(- {\pi \over 2} -\theta) = -\sin({\pi \over 2} + \theta)$
类似地$\cos(\pi - \theta) = \cos({\pi \over 2} - (- {\pi \over 2} + \theta)) = \sin(- {\pi \over 2} + \theta) = -\sin({\pi \over 2} - \theta)$
注意到$\cos(\pi + \theta) = \cos(-\pi + \theta) = \cos(\pi - \theta)$，我们就得到了 $$\sin({\pi \over 2} + \theta) = \sin({\pi \over 2} - \theta)$$ 用类似地思路，还可以证明以下等式 $$\cos({\pi \over 2} + \theta) = -\cos({\pi \over 2} - \theta)$$ 这样就解释了上表中函数值关于$\pi \over 2$的对称性。
基本上来说，我们只需要记住$(0,{\pi \over 2})$范围内的正弦函数值，其余范围的各函数值均可由上述性质推得。

三角函数：恒等式

周期性： \begin{eqnarray} \sin(\theta \pm 2\pi) = \sin(\theta) \\ \cos(\theta \pm 2\pi) = \cos(\theta) \end{eqnarray} 关于0的对称性： \begin{eqnarray} \sin(0+\theta) = -\sin(0-\theta) \\ \cos(0+\theta) = \cos(0-\theta) \\ \tan(0+\theta) = -\tan(0-\theta) \end{eqnarray} 关于$\pi \over 2$的对称性： \begin{eqnarray} \sin({\pi \over 2} + \theta) = \sin({\pi \over 2} - \theta) \\ \cos({\pi \over 2} + \theta) = -\cos({\pi \over 2} - \theta) \label{dc1} \end{eqnarray} 关于$\pi$的对称性： \begin{eqnarray} \sin(\pi + \theta) = -\sin(\pi - \theta) \\ \cos(\pi + \theta) = \cos(\pi - \theta) \end{eqnarray} 综合使用以上公式，可以帮助我们把任意角的三角函数转换到$[0,{\pi \over 2}]$范围内。
正余互换： \begin{eqnarray} \sin(\alpha) = \cos({\pi \over 2} - \alpha) \label{zy1} \\ \cos(\alpha) = \sin({\pi \over 2} - \alpha) \label{zy2} \\ \csc(\alpha) = \sec({\pi \over 2} - \alpha) \label{zy3} \\ \tan(\alpha) = \cot({\pi \over 2} - \alpha) \label{zy4} \end{eqnarray} \eqref{zy2}由\eqref{zy1}经代换$\alpha \to {\pi \over 2} - \alpha$得到。\eqref{zy3}由\eqref{zy1}取倒数得到。\eqref{zy4}由\eqref{zy1} \eqref{zy2}做除法得到。 勾股定理： \begin{eqnarray} \sin^2(\theta) + \cos^2(\theta) = 1 \\ \tan^2(\theta) + 1 = \sec^2(\theta) \label{gg2} \\ 1 + \cot^2(\theta) = \csc^2(\theta) \label{gg3} \end{eqnarray} 综合使用上述公式，能让我们将6个三角函数中的任意一个，用其他五个中的任意一个表示出来。
角度的和与差： \begin{eqnarray} \sin(\alpha + \beta) = \sin(\alpha)\cos(\beta) + \cos(\alpha)\sin(\beta) \label{\sin1} \\ \cos(\alpha + \beta) = \cos(\alpha)\cos(\beta) - \sin(\alpha)\sin(\beta) \label{\cos1} \end{eqnarray} 把\eqref{\sin1}\eqref{\cos1}中的$\beta$替换成$-\beta$可以得到对应的角度差公式。 \begin{eqnarray} \sin(\alpha - \beta) = \sin(\alpha)\cos(\beta) - \cos(\alpha)\sin(\beta) \label{\sin2} \\ \cos(\alpha - \beta) = \cos(\alpha)\cos(\beta) + \sin(\alpha)\sin(\beta) \label{\cos2} \end{eqnarray} 公式\eqref{\sin1}～\eqref{\cos2}可以有几种证明方法，最快的方法是借助欧拉公式。 $$(\cos(\alpha) + \mathrm{i}\sin(\alpha))(\cos(\beta) + \mathrm{i}\sin(\beta)) = \mathrm{e}^{\mathrm{i}\alpha}\mathrm{e}^{\mathrm{i}\beta} = \mathrm{e}^{\mathrm{i}(\alpha + \beta)} = \cos(\alpha + \beta) + \mathrm{i}\sin(\alpha + \beta)$$ 将上式最左边乘开，并令其与最右端实部虚部分别相等即可得到。(用欧拉公式似乎有循环论证的嫌疑，因为欧拉公式的证明需要用幂级数展开，这就涉及到求三角函数的导数，而三角函数的导数正是通过角度和公式进行计算得出的...)
将\eqref{\sin1}\eqref{\cos1}相除并化简还可得到 \begin{equation} \tan(\alpha + \beta) = {\tan(\alpha) + \tan(\beta) \over 1 - \tan(\alpha)\tan(\beta)} \end{equation} 将\eqref{\sin1}\eqref{\sin2}相加减，我们就得到函数乘积公式： \begin{eqnarray} \sin(\alpha)\cos(\beta) = {\sin(\alpha + \beta) + \sin(\alpha - \beta) \over 2} \\ \cos(\alpha)\sin(\beta) = {\sin(\alpha + \beta) - \sin(\alpha - \beta) \over 2} \end{eqnarray} 将\eqref{\cos1}\eqref{\cos2}相加减，我们就得到函数乘积公式： \begin{eqnarray} \sin(\alpha)\sin(\beta) = {\cos(\alpha - \beta) - \cos(\alpha + \beta) \over 2} \\ \cos(\alpha)\cos(\beta) = {\cos(\alpha - \beta) + \cos(\alpha + \beta) \over 2} \end{eqnarray} 做变换$\theta = \alpha + \beta$，$\phi = \alpha - \beta$，故有$\alpha = {\theta + \phi \over 2}$，$\beta = {\theta - \phi \over 2}$。
这将方便我们导出如下的函数和差公式 \begin{eqnarray} \sin(\theta) + \sin(\phi) = 2\sin({\theta + \phi \over 2})\cos({\theta - \phi \over 2}) \\ \sin(\theta) - \sin(\phi) = 2\cos({\theta + \phi \over 2})\sin({\theta - \phi \over 2}) \\ \cos(\theta) + \cos(\phi) = 2\cos({\theta + \phi \over 2})\cos({\theta - \phi \over 2}) \\ \cos(\theta) - \cos(\phi) = -2\sin({\theta + \phi \over 2})\sin({\theta - \phi \over 2}) \end{eqnarray} 角度和公式的另一个有用的特例是2倍角和3倍角公式 \begin{eqnarray} \sin(2\theta) = 2\sin(\theta)\cos(\theta) \label{\sin5} \\ \begin{split} \cos(2\theta)& = \cos^2(\theta) - \sin^2(\theta)\\ & = 1 - 2\sin^2(\theta) \\ & = 2\cos^2(\theta) - 1 \end{split} \label{\cos5} \\ \sin(3\theta) = 3\sin(\theta) - 4\sin^3(\theta) \\ \cos(3\theta) = 4\cos^3(\theta) - 3\cos(\theta) \end{eqnarray} 这将方便我们导出如下的函数降幂公式： \begin{eqnarray} \sin^2(\theta) = {1 - \cos(2\theta) \over 2} \label{\sin3} \\ \cos^2(\theta) = {1 + \cos(2\theta) \over 2} \label{\cos3} \\ \sin^3(\theta) = {3\sin(\theta) - \sin(3\theta) \over 4} \label{\sin4} \\ \cos^3(\theta) = {3\cos(\theta) + \cos(3\theta) \over 4} \label{\cos4} \end{eqnarray} 组合使用公式\eqref{\sin3}～\eqref{\cos4}，使我们能够将4次5次甚至更高次幂降下来。
使用2倍角公式的一个例子是导出所谓的“万能公式” \begin{eqnarray} \sin(\theta) = {2\tan({\theta \over 2}) \over 1 + \tan^2({\theta \over 2})} \\ \cos(\theta) = {1 - \tan^2({\theta \over 2}) \over 1 + \tan^2({\theta \over 2})} \\ \tan(\theta) = {2\tan({\theta \over 2}) \over 1 - \tan^2({\theta \over 2})} \end{eqnarray} 只需将\eqref{\sin5}\eqref{\cos5}的右端分子分母同时除以$\cos^2(\theta)$，再将$\theta$替换为$\theta \over 2$即可得到。
另一个例子是求$\pi \over 6$的正弦值，只需将$\theta = {\pi \over 6}$带入3倍角公式，得$3\sin({\pi \over 6}) - 4\sin^3({\pi \over 6}) = \sin(3{\pi \over 6}) = \sin({\pi \over 2})= 1$，即 $$3\sin({\pi \over 6}) - 4\sin^3({\pi \over 6}) - 1 = 0$$ 解上述方程，如果观察到-1是其中一根，可分解因式为 $$4(\sin({\pi \over 6}) + 1)(\sin({\pi \over 6}) - {1 \over 2})^2 = 0$$ 即可解得$\sin({\pi \over 6}) = {1 \over 2}$。用一下勾股定理还可得到$\cos({\pi \over 6}) = {\sqrt{3} \over 2}$

反三角函数：恒等式

记$x = \sin(y)$,则$\csc(y) = {1 \over \sin(y)} = {1 \over x}$。由于$y = \arcsin(\sin(y)) = \arccsc(\csc(y))$，我们就得到 \begin{equation} \arcsin(x) = \arccsc({1 \over x}) \end{equation} 类似地，可得 \begin{eqnarray} \arcsec(x) = \arccos({1 \over x}) \\ \arccot(x) = \arctan({1 \over x}) (x > 0) \end{eqnarray} 记$x = \sin(y)$,则$\sin(-y) = -\sin(y) = -x$。由于$y = \arcsin(\sin(y)) = -\arcsin(\sin(-y))$,我们就得到 \begin{equation} \arcsin(x) = -\arcsin(-x) \end{equation} 类似地，可得 \begin{eqnarray} \arccsc(x) = -\arccsc(-x) \\ \arctan(x) = -\arctan(-x) \end{eqnarray} 注意到$\arccos$的值域，记$x = \cos(y) = \cos({\pi \over 2} - ({\pi \over 2} - y))$, 根据\eqref{dc1},有
$\cos({\pi \over 2} - ({\pi \over 2} - y)) = -\cos({\pi \over 2} + ({\pi \over 2} - y)) = -\cos(\pi - y)$,
总之，我们得到了$x = \cos(y) = -\cos(\pi - y)$。
由于$y = \arccos(\cos(y)) = \pi - \arccos(\cos(\pi - y))$,我们就得到 \begin{equation} \arccos(x) = \pi - \arccos(-x) \end{equation} 类似地，由于$\sec(y) = -\sec(\pi - y)$以及$\cot(y) = -\cot(\pi - y)$,可以得到 \begin{eqnarray} \arcsec(x) = \pi - \arcsec(-x) \\ \arccot(x) = \pi - \arccot(-x) \end{eqnarray} 基于公式\eqref{zy1} \eqref{zy3} \eqref{zy4}，仿照上述证明方法，还可以得到 \begin{eqnarray} \arcsin(x) = {\pi \over 2} - \arccos(x) \\ \arccsc(x) = {\pi \over 2} - \arcsec(x) \\ \arctan(x) = {\pi \over 2} - \arccot(x) \end{eqnarray}
